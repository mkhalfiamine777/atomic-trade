# 📄 تقرير الصفحات الثمانية الجديدة

## 📋 نظرة عامة

| # | الصفحة | المسار | الوظيفة |
|---|--------|--------|---------|
| 1 | Feed | `/feed` | التغذية الرئيسية (For You) |
| 2 | Upload | `/upload` | رفع فيديو جديد |
| 3 | Stories | `/stories` | عرض القصص |
| 4 | Create Story | `/stories/create` | إنشاء قصة |
| 5 | Live List | `/live` | قائمة البث المباشر |

## 1️⃣ صفحة Feed (`/feed`)

### 🎯 الوظيفة
الصفحة الرئيسية للتصفح، تعرض فيديوهات قصيرة بأسلوب TikTok (تمرير عمودي).

### 🖼️ التصميم
```
┌─────────────────────────────┐
│ ┌─────────────────────────┐ │
│ │                         │ │
│ │     📹 فيديو ملء       │ │
│ │       الشاشة           │ │
│ │                         │ │
│ │ ┌───┐                   │ │
│ │ │👤│ اسم البائع         │ │
│ │ └───┘ @username         │ │
│ │ 📝 وصف المنتج...        │ │
│ │ 🏷️ #هاشتاغ             │ │
│ └─────────────────────────┘ │
│         ⬇️ تمرير للأسفل     │
│ ┌───┐                       │
│ │ ❤️│ 1.2K                  │
│ │ 💬│ 234                   │
│ │ 🔗│                       │
│ │ 🔖│                       │
│ │ 🛒│ شراء                  │
│ └───┘                       │
└─────────────────────────────┘
```

### ⚙️ الميزات
- Infinite Scroll (تمرير لانهائي)
- تشغيل تلقائي عند الظهور
- إيقاف عند الخروج من العرض
- أزرار التفاعل على اليمين
- رابط مباشر للمنتج

├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │                         │ │
│ │   📁 اسحب الفيديو هنا   │ │
│ │     أو اضغط للرفع       │ │
│ │                         │ │
│ │   🎥 تسجيل مباشر        │ │

├─────────────────────────────┤

│ │ أكتب وصفاً للفيديو...   │ │

## 📊 ملخص التقنيات لكل صفحة

| الصفحة | المكونات الرئيسية | الـ APIs |
|--------|-------------------|----------|
| Feed | VideoFeed, VideoCard | GET /videos |
| Upload | VideoRecorder, VideoUploader | POST /videos |
| Stories | StoryBar, StoryViewer | GET /stories |
| Create Story | StoryCreator | POST /stories |
| Live List | LiveCard Grid | GET /live |
| Watch Live | LivePlayer, LiveChat | WS /live/[id] |
| Start Live | LiveControls | POST /live/start |
| Profile | ProfileHeader, VideoGrid | GET /users/[id] |
