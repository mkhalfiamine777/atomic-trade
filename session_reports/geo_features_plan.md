# خطة تنفيذ الميزات الجغرافية (Geo-Features)

## 1. أيقونات مخصصة (Custom Markers) 📍
سنقوم باستبدال الدبوس الأزرق الافتراضي بأيقونات تعبر عن الهوية:
- **👤 المستخدم (أنت):** أيقونة دائرية مع صورة الملف الشخصي (Avatar).
- **🏪 المتجر (Shop):** أيقونة متجر مميزة.
- **👑 مول الحومة (Lord):** أيقونة تاج ذهبي للمسيطر على المنطقة.

## 2. منطق الـ 300 متر (The 50m Logic) 📏
سنقوم بإنشاء `hook` مخصص باسم `useGeolocation` يقوم بـ:
1.  تتبع موقع المستخدم الحي `navigator.geolocation.watchPosition`.
2.  حساب المسافة بين المستخدم وجميع المتاجر المجاورة باستخدام "Haversine Formula".
3.  إطلاق "تنبيه" (Notification/Sound) عند الدخول في دائرة قطرها 300 متر حول متجر.

## 3. محاكاة البيانات (Dummy Data) 🧪
بما أننا لا نملك مستخدمين حقيقيين بعد، سنضيف زر "Add Test Shop" يضع متجراً وهمياً بجانب موقعك الحالي لتجربة التنبيه فوراً.
