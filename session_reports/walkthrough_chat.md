# دليل اختبار نظام الدردشة الآمنة (Secure Chat)

## 1. إعادة تشغيل السيرفر
الخطوة الأولى والأهم: بما أننا قمنا بتحديث قاعدة البيانات، يجب إعادة تشغيل السيرفر ليعمل كل شيء بشكل صحيح.
- في الترمينال، اكتب: `npm run dev`

## 2. السيناريو (كيف تختبر؟)
لكي تختبر المحادثة، تحتاج إلى مستخدمين اثنين (بائع ومشتري).

### الطريقة:
1.  **افتح متصفح المتخفي (Incognito):** وسجل الدخول بحساب جديد (مختلف عن حسابك الحالي).
2.  **أضف طلباً أو منتجاً:** في الحساب الجديد، أضف "منتجاً" أو "طلباً".
3.  **عُد لحسابك الأصلي:** انظر للخريطة، ستجد أيقونة الحساب الجديد.
4.  **ابدأ الدردشة:** اضغط على الأيقونة -> ثم اضغط **"💬 دردشة آمنة"**.
5.  **جرب المراسلة:** ستظهر نافذة الدردشة فوراً.

## المميزات الجديدة
- **الخصوصية:** لا يتم كشف أرقام الهواتف.
- **التواصل الفوري:** الرسائل تظهر للطرفين (يتم التحديث كل 2 ثانية).
- **واجهة مريحة:** نافذة منبثقة لا تغطي الخريطة بالكامل.
