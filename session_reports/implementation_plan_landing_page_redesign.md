# 🎨 خطة إعادة تصميم الصفحة الرئيسية (Landing Page Redesign)

## 🎯 الهدف
تحويل الخريطة إلى **خلفية حية كاملة** (Full-screen Background) للصفحة الرئيسية، مع تحسين الإضاءة لتبدو أكثر حيوية.

## 🛡️ استراتيجية الأمان (Revert Policy)
قبل أي تعديل، سنقوم بإنشاء "نقطة استعادة" (Commit) لضمان إمكانية الرجوع للوضع السابق بضغطة واحدة.

## 🛠️ تفاصيل التنفيذ

### 1. جعل الخريطة خلفية (The Canvas)
سنستخدم **صورة الخريطة الثابتة** (Static Image) كخلفية لكامل الصفحة `body` أو `main`:
*   **المصدر:** صورة عالية الجودة (Unsplash) تشبه الخرائط الليلية أو الرمادية.
*   **الموقع:** `fixed inset-0 z-0 object-cover` (لتغطية الشاشة بالكامل).

### 2. تعديل الإضاءة (The Lighting) 💡
لجعل الصورة الخلفية أكثر إشراقاً وأقل كآبة:
*   **CSS Filter:** سنطبق فلتر تفتيح:
    ```css
    filter: brightness(1.1) contrast(0.9) saturate(1.1);
    ```
    *زيادة السطوع قليلاً وتقليل التباين لتبقى النصوص مقروءة.*

### 3. تعديل المحتوى (The Overlay)
بما أن الخلفية أصبحت مشغولة (Busy Background)، يجب تعديل النصوص لتكون قابلة للقراءة:
*   **Hero Section:** إضافة خلفية زجاجية (Glassmorphism): `bg-background/60 backdrop-blur-md`.
*   **الأزرار:** التأكد من تباين الألوان (High Contrast).

## 📝 الخطوات البرمجية
1.  تعديل `src/app/page.tsx`:
    *   إزالة `img` القديمة.
    *   إضافة `<Map />` كخلفية.
2.  تطبيق مؤثرات `filter` في `globals.css` أو `Tailwind classes`.
