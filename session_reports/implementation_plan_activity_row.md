# 📋 خطة تنفيذ: شريط الأنشطة (Activity Row)

## 🎯 الهدف
عرض أيقونات الأنشطة (الطلبات 📣، القصص 📸، المنتجات 📦) في **صف أفقي منظم أسفل أيقونة المستخدم** بدلاً من انتشارها دائرياً أو عشوائياً.

## 🎨 التصور (Visual Concept)
عندما يكون هناك مستخدم (فرد/متجر/شركة) لديه أنشطة متعددة في نفس الموقع:
1.  تظهر **الأيقونة الرئيسية** (النيون) في المركز.
2.  عند النقر أو التفاعل، تنبثق **الأيقونات الفرعية** في صف واحد مستقيم أسفل الأيقونة الرئيسية.

```text
       [ 👤 ]  <-- أيقونة المستخدم (المركز)
         |
   [📣] [📸] [📦]  <-- شريط الأنشطة (يظهر أسفلها)
```

## 🛠️ التغييرات التقنية

### 1. تعديل `SmartMarkerGroup.tsx`

سنقوم بتغيير خوارزمية توزيع الأيقونات (`Spiderfy Logic`) من دائرة إلى **خط مستقيم**.

#### الخوارزمية الجديدة (Layout Logic):
بدلاً من حساب زاوية دائرية (`angle`):
*   **خط العرض (Latitude):** نقم بإنقاصه قليلاً (لتحريك الصف لأسفل).
*   **خط الطول (Longitude):** نوزع الأيقونات يميناً ويساراً بناءً على عددها.

```typescript
// دالة حساب الموقع الجديد
const getRowPosition = (index: number, total: number) => {
    const rowOffsetLat = -0.00015; // إزاحة للأسفل (~15 متر)
    const itemSpacingLng = 0.00018; // مسافة بين كل أيقونة (~18 متر)
    
    // وضع الأيقونات في المنتصف (Centering)
    const startLng = -(total - 1) * itemSpacingLng / 2;
    const currentLngOffset = startLng + (index * itemSpacingLng);

    return [
        center[0] + rowOffsetLat, 
        center[1] + currentLngOffset
    ];
}
```

### 2. تحسينات المظهر (Styling)
*   إضافة خط وصل (اختياري) أو مجرد تقارب بصري.
*   ضمان أن الأيقونات الفرعية أصغر قليلاً أو بنفس الحجم (حسب الرغبة).

## ✅ الفوائد
1.  **تنظيم أفضل:** عدم حجب الخريطة بدائرة كبيرة.
2.  **سهولة الوصول:** الأيقونات قريبة من المستخدم وتدل على تبعيتها له.
3.  **مظهر عصري:** يتناسب مع تصميم النيون الجديد.
