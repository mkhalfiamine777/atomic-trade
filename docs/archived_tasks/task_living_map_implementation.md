## 🎯 الهدف (Objective)
تحويل الخريطة من مجرد عارض للمواقع إلى سطح مكتب تفاعلي ينبض بالحياة، يربط بين العالم الافتراضي والواقعي عبر:
1.  **الوسم الجغرافي (Geo-tagging):** لكل تفاعل.
2.  **العلامات الذكية (Smart Markers):** تعرض المحتوى مباشرة.
3.  **التفاعل الزجاجي:** عند النقر.
