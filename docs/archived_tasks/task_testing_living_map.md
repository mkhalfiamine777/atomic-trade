# 🧪 خطة اختبار الخريطة الحية (Living Map Testing Plan)
**الحالة:** قيد التحضير 🚧
**تاريخ الإنشاء:** 11-02-2026

---

## 🎯 الهدف (Objective)
التحقق من صحة عمل ميزة **الخريطة الحية (Living Map)** و **الوسم الجغرافي (Geo-tagging)** قبل الانتقال للمرحلة التالية.

---

## ✅ 1. الاختبارات الممكنة (Feasible Tests)
هذه الاختبارات يمكن إجراؤها فوراً على جهاز التطوير (Localhost):

### **أ. إنشاء المحتوى (Content Creation)**
- [x] **إذن الموقع:** هل يطلب المتصفح الإذن عند فتح نافذة "إنشاء منشور"؟
- [x] **التقاط الإحداثيات:** هل تظهر رسالة "تم تحديد الموقع ✅"؟
- [x] **الإرسال:** هل يتم حفظ المنشور في قاعدة البيانات مع حقول `latitude` و `longitude` غير فارغة؟
- [x] **المعالجة:** هل تظهر رسالة نجاح (Toast) وتختفي النافذة؟

### **ب. عرض الخريطة (Map Display)**
- [x] **ظهور العلامة (Marker Visibility):** هل تظهر علامة المنشور الجديد في الموقع الصحيح؟
- [x] **رمز العلامة (Icon):** هل الأيقونة مميزة (دائرية مع صورة مصغرة)؟
- [x] **نافذة المعلومات (Popup):** عند النقر، هل تظهر النافذة الزجاجية؟
    - [x] الصورة واضحة؟
    - [x] اسم المستخدم والصورة الرمزية يظهران؟
    - [x] النص (Caption) يظهر بشكل صحيح؟

### **ج. التفاعل (Interaction)**
- [x] **زر الملف الشخصي:** هل ينقلني الزر "👤 الملف" إلى صفحة المستخدم (`/u/[username]`)؟
- [x] **الإعجاب (Like):** هل يظهر إشعار (Toast) عند النقر على "❤️ إعجاب" (حتى لو لم يحفظ في DB بعد)؟
- [x] **القائمة العائمة (FAB):**
    - [x] **الزر البنفسجي (منشور):** يفتح نافذة "منشور اجتماعي"؟
    - [x] **الزر الوردي (قصة):** يفتح نافذة "قصة جديدة"؟
    - [x] **الزر الأزرق (منتج):** يفتح نافذة "إضافة منتج"؟

### **د. اقتصاد الثقة (Trust Economy) - *New***
- [x] **شارة الثقة (Trust Badge):** هل تظهر 🛡️ للمستخدمين ذوي السمعة > 80؟
- [x] **الشفافية (Opacity):** هل تظهر منشورات المستخدمين الجدد (السمعة < 50) بشفافية 0.7؟
- [x] **أداة المطور:** هل يعمل رابط API لتغيير السمعة (`/api/dev/set-rep`)؟

## ⚠️ 2. الاختبارات المحدودة (Limited / Simulated Tests)
تتطلب ظروفاً خاصة أو محاكاة:

- [ ] **سيناريو رفض الموقع:** ماذا يحدث لو ضغط المستخدم "Block Location"؟
    - *المتوقع:* تظهر رسالة خطأ ❌، ويتم النشر بدون موقع (أو يُمنع النشر حسب المنطق).
- [ ] **دقة GPS:** هل الإحداثيات دقيقة مقارنة بموقعك الفعلي على الخريطة؟
    - *ملاحظة:* الأجهزة المكتبية تستخدم IP Geolocation وقد تكون غير دقيقة (±5 كم).

---

## ❌ 3. الاختبارات غير الممكنة حالياً (Not Feasible Yet)
تعتمد على ميزات مستقبلية:

- [ ] **تصفية الثقة (Trust Filtering):** إخفاء المنشورات من المستخدمين الجدد/المشبوهين (المرحلة 4).
- [ ] **التحديث اللحظي (Real-time Sync):** ظهور المنشور فوراً عند مستخدم آخر (يتطلب Socket.io - المرحلة 2).
- [ ] **التجميع الذكي (Clustering Animation):** اختبار أداء تجميع 1000+ علامة (يتطلب بيانات وهمية ضخمة).

## 🛠️ خطوات التنفيذ (Execution Steps)
1.  افتح المتصفح على `http://localhost:3000/dashboard`.
2.  اضغط زر (+) لإنشاء منشور.
3.  وافق على إذن الموقع.
4.  اختر صورة واكتب نصاً.
5.  اضغط "نشر".
6.  اذهب للخريطة وابحث عن العلامة الخاصة بك.
